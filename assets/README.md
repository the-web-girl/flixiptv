# Assets
Ce dossier contiendra les images et polices de l'application.

## Fonts
Téléchargez Outfit depuis Google Fonts :
https://fonts.google.com/specimen/Outfit

Placez les fichiers .ttf ici :
- Outfit-Regular.ttf
- Outfit-SemiBold.ttf
- Outfit-Bold.ttf

## Images
Ajoutez ici vos icônes et images (logo.png, placeholder.png, etc.)
